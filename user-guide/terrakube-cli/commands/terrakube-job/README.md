# terrakube job

This command consists of multiple subcommands to interact with jobs. It can be used to create and list teams.

#### Usage

```
terrakube job [command]
```

#### Commands

| Command                   | Description  |
| ------------------------- | ------------ |
| [create](job-create.md)   | create a job |
| [list](job-list.md)       | list jobs    |
